## 🧰 Setup Snippet

```
pip install mkdocs
mkdocs new my-docs
cd my-docs
# Apply the NexAscend theme bundle
cp -r mkdocs-theme-bundle/* .
mkdocs serve
```

⚡ Preview your decentralized docs on `localhost:8000`
